This package is a comprehensive collection of more than 100 3D items, animated characters, props, effects, ui elements and  and tiling textures.
Ideal for rapid mobile game prototyping and any kind of games with a cartoon- or handpainted style.
The package comes with 80+ different 3D items in FBX format, ready to use with materials and textures assigned. Package also contains particle effects, most common UI elements used in mobile games and some tiling textures for terrains.
You can use it right off the bat. You get also the original texture source files in Adobe® Photoshop® format with all layers intact and UVW guidelines included.
And since this is a mobile game model pack, all models have poly count of no more than 1000 polys.
